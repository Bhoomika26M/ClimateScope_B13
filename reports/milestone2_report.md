Objective:

To perform advanced exploratory data analysis (EDA) on the cleaned climate dataset and develop a fully interactive web-based climate analytics dashboard.

Technologies Used:-

->Python
->Pandas
->Streamlit
->Plotly
->Scikit-learn
->PyCountry

Implemented Features

1.Interactive Multi-Tab Dashboard
->Structured UI using Streamlit Tabs
->Clean and user-friendly layout

2.Multi-Level Time Aggregation
->Daily
->Monthly
->Yearly
3.Country Trend Analysis

->Temperature trend visualization
->Optional 7-period Moving Average smoothing

4.Country Comparison
->Side-by-side temperature comparison
->Dynamic filtering

5.Global Temperature Map
->Choropleth world map
->ISO-3 country mapping
->Gradient-based visualization

6.Correlation Analysis
->Heatmap of climate feature relationships
->Statistical insight visualization

7.Automated Climate Insights
->Hottest country detection
->Coldest country detection
->Worst air quality detection

8.Machine Learning Integration
Linear Regression Model
->Temperature prediction based on humidity
->Interactive slider input
