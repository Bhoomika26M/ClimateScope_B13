# 🌍 ClimateScope: Global Weather Analytics Dashboard
**Developed by: Mudragada Swapna**

### Project Overview
An interactive Streamlit dashboard to analyze global weather patterns using the **Global Weather Repository** dataset.

### Folder Structure
- `src/`: Contains the source code (`app.py`).
- `data/`: Contains the dataset (`GlobalWeatherRepository.csv`).
- `requirements.txt`: List of required Python libraries.
