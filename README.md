# ClimateScope_B13

🌍 ClimateScope: Climate Intelligence Dashboard

📌 Project Overview

ClimateScope is an end-to-end data analytics project designed to analyze global weather patterns and provide meaningful insights through an interactive dashboard.

This project demonstrates skills in:

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Statistical Analysis
- Data Visualization
- Dashboard Development using Streamlit


🚀 Features

🔹 Milestone 1 - Data Processing

- Cleaned raw weather dataset
- Handled missing values
- Created new features (Year, Month)
- Generated monthly aggregated dataset

🔹 Milestone 2 - Core Analysis

- Statistical summary of climate data
- Correlation analysis between weather parameters
- Seasonal temperature trend analysis
- Extreme weather event detection (Top 5%)
- Country-wise temperature comparison

🔹 Milestone 3 - Interactive Dashboard

- Real-time filtering (Country, Date, Metric)
- KPI metrics (Temperature, Humidity, Wind, Rainfall)
- Climate trend visualizations
- Extreme weather detection charts
- Global temperature map (Choropleth)
- Dataset download option


🛠️ Tech Stack

- Python
- Pandas & NumPy
- Plotly
- Streamlit
